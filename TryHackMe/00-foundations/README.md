# 00-Foundations

This track is the onboarding shelf for the public TryHackMe corpus.

Use it when you want to build the baseline that makes the rest of the repo easier to read:

- core security vocabulary
- command-line habits
- note-taking and research workflow
- beginner tooling concepts
- transferable fundamentals before deeper web, networking, or blue-team tracks

## Summary

This page is a lightweight directory for the `00-foundations` track. Use it to find the best entry point, browse the main sub-areas, and jump into the foundational notes that support the rest of the public TryHackMe corpus.
