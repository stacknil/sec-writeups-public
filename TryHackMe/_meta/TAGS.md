Use this page as the controlled-vocabulary reference for TryHackMe note front matter.

> Prefer fewer tags per note when possible.

## Hygiene rules

- Avoid inventing new values unless the taxonomy cannot express the note accurately.
- Use `TARGET_IP`, `USER_A`, and `example.com` style placeholders in public notes.
- Keep write-ups sanitized; do not publish full exploit chains or copy-paste payload walkthroughs.
