# Support

## This repository is not a general support channel

`sec-writeups-public` is a public documentation repository for sanitized security notes and writeups.

It does **not** provide:

- general technical support
- target-specific help
- exploit help
- private lab assistance
- off-repository consulting

## Use the correct issue type

- **Content correction**: factual error, wording problem, misleading explanation, incorrect command context, or note-level fix
- **Publication safety concern**: possible sensitive detail, failed sanitization, dangerous publication boundary issue, or placeholder failure
- **Structure / taxonomy improvement**: note organization, metadata, naming, taxonomy, front matter, or classification improvements

## Best-effort maintenance

Review and response are best-effort.

Well-scoped, file-specific, repository-relevant reports are easier to triage and act on.
